# تطبيق مبيعاتي 📊

تطبيق ويب متقدم لتتبع المبيعات وإدارة الأداء - مصمم خصيصاً لفريق المبيعات.

## 🚀 الميزات الرئيسية

### 📱 التطبيق
- **13 هدف مبيعات** برموز مختلفة
- **إدخال يومي** للمبيعات بقيمة وعدد وأمر البيع
- **3 أنواع تقارير**: يومي، أسبوعي، شهري
- **تقرير أوامر** منفصل
- **رسوم بيانية** ملونة وسهلة الفهم
- **البحث بالتاريخ** والتنقل بين الأيام

### 🎯 التصدير
- **تصدير WhatsApp**: انسخ والصق التقرير مباشرة
- **تصدير PDF**: طباعة احترافية
- **تصدير JSON**: للأرشفة والنسخ الاحتياطية

### 👔 وضع المشرف
- **بدون كلمة مرور** - دخول فوري
- **استيراد التقارير** من موظفيه تلقائياً
- **لوحة الفريق** - عرض جميع الموظفين
- **تقرير PDF للفريق** - تفاصيل كل موظف

### 📊 لوحة التحكم
- **كلمة مرور**: 1234
- **إحصائيات شاملة** - أداء جميع المشرفين
- **تقرير PDF** - مفصل لكل الموظفين والمشرفين
- **تجميع البيانات** من عدة مشرفين

## 📥 التثبيت

### الطريقة 1: بدون تثبيت
افتح الرابط مباشرة في المتصفح:
```
https://next-mysales.github.io/my-sales/
```

### الطريقة 2: تثبيت على iPhone
1. افتح Safari (ليس Chrome)
2. اكتب الرابط أعلاه
3. اضغط زر المشاركة (↑)
4. اختر "Add to Home Screen"
5. أكتب: مبيعاتي
6. اضغط "Add"

### الطريقة 3: تثبيت على Android
1. افتح Chrome
2. اكتب الرابط أعلاه
3. اضغط النقاط الثلاث (⋮)
4. اختر "Install app"
5. اضغط "Install"

## 📊 حساب التقارير

### التقرير اليومي
```
المبيعات = مبيعات اليوم الواحد فقط
الهدف = الهدف اليومي
النسبة = (المبيعات ÷ الهدف) × 100
```

### التقرير الأسبوعي
```
المبيعات = مبيعات 7 أيام (أحد إلى سبت)
الهدف = الهدف اليومي × 7
النسبة = (المبيعات ÷ الهدف) × 100
```

### التقرير الشهري
```
المبيعات = مبيعات الشهر الكامل (1-31)
الهدف = الهدف اليومي × أيام الشهر
النسبة = (المبيعات ÷ الهدف) × 100
```
